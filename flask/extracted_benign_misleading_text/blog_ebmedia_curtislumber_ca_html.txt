DNS A record meaning:
The A record (Address Record) is one of the most commonly used DNS records. It points a domain or subdomain to IP address. In other words, the A record specifies the IP address (IPv4) for a given host.

The DNS A record is one of the fundamental types of DNS records. Remember that creating an A record, you can only point to an IPv4-address. You need to create an AAAA record if you want to point to IPv6-address.
