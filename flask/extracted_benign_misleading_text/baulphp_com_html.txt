MySQL. La base de datos
¿Qué es MySQL Server?
La base de datos MySQL es una de las base de datos más populares en el mundo del internet y abarca una gran cuota de mercado. Sin embargo, podemos dar el concepto de MySQL, como un gestor y/o administrador de base de datos relacional, multiusuario. Su uso se caracteriza por manejar grandes cantidades de información de neustros sistemas, hoy en día muchos proyectos e ideas de internet como Facebook o Twitter nacieron en base de este núcleo.
Características de MySQL
Este gestor llamado MySQL, es una base de datos polivalente. Es decir puede funcionar en varios sistemas operativos como por ejemplo Linux, Windows, Mac, etc., y en la actualizada por su gran uso existe gran número de componentes para conectarse a ella desde varios lenguajes de programación incluidos el PHP.
La creación de esta importante base de datos fue posible usando una mezcla entre varios lenguajes como ser: C y C++. En la actualidad hablar de base de datos en un servidor web, es hablar de MySQL. Y es complemento perfecto para sistemas creados con PHP. Algunas de las ventajas del uso de este gestor son las siguientes:
- Es Multiplataforma: Funciona en diversos sistemas operativos como ser: Linux, Windows, AIX, Solaris, y demás sistemas que lo soportan.
- Múltiples motores de almacenamiento. Su sencillez porque se adaptan a las distintas necesidades del desarrollador: entre los motores más conocidos tenemos MyISAM, InnoDB, Memory, etc.
- La velocidad para realizar una tarea y a la hora de realizar operaciones.
- Su arquitectura soporta una amplia gama de tipos de datos.
- Tiene una gran comunidad de desarrolladores que la respaldan.
- Posee una buena capacidad de indexación y búsqueda y uso de fulltext, con la cual podemos realizar búsquedas complejas.

- Manipular el árbol DOM de manera eficiente.
- Manejar eventos dentro del navegador muy fácilmente.
- generar animaciones en nuestra página web.
- AJAX que interactúa con el lenguaje PHP para realizar acciones ligadas con el CRUD MySQL.
