Xobor-Foren sind individuell zu gestalten, leicht zu bedienen und müssen nicht selbst installiert werden.
Ein eigenes Forum selbst erstellen klappt nun auch ohne Programmierkenntnisse - wer sich allerdings in HTML und Co. auskennt, hat die Möglichkeit ein eigenes Template individuell bearbeiten zu können.
Individuell und leicht verständlich
Durch Updates der Software werden regelmäßig neue Extras eingespielt, wie zum Beispiel ein Portal, Chat, Bildergalerie, Kalender, Google Maps oder einen Blog.
In unserem Plugin-Store finden Sie eine große Auswahl an Erweiterungen. Plugins werden nicht nur von Xobor entwickelt sondern können auch von anderen Administratoren zur Verfügung gestellt werden.
Extras und Plugins
Nutzen Sie die Auswahl von Designvorlagen aus unseren Foren-Designs. In Ihrer Administration können Sie jedes Design individuell bearbeiten und dabei eigene Grafiken einsetzen - ganz ohne Programmierkenntnisse. Wenn Sie Erfahrung haben mit HTML & CSS können Sie auch Änderungen im Template vornehmen, bei Xobor ist der Kreativität keine Grenzen gesetzt.
Vielfältige Designs
Mit unserem Mobile Template haben Sie speziell für User die Ihr Forum mit einem Smartphone besuchen eine eigene Version. Das Template ist übersichtlich und für jedes Mobile-Endgerät maßgeschneidert. Auch bei größeren Foren behält man so besser den Überblick und profitiert auch mit schnelleren Ladegeschwindigkeiten am Smartphone.
