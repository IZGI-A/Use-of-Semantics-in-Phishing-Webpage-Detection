Auf diesen Windows Server, haben Sie die Möglichkeit für kleines Geld:
- Web - Speicherplatz für Webseiten-Hosting (Zugang über FTP)
- Webspace in verschiedenen Größen
- Welten Hosting für eure Welten in der "Activeworlds 3D Plattform"
zu erwerben.
Für ihren Webspace wählen Sie bitte im "Shop" das für Sie passende Paket, dies wird Ihnen dann mit ihrer eigenen Wunschdomain bereitgestellt. Sollten sie keine eigene Domain benötigen, wird die Server eigene Domain verwendet (ihrname.heikowolf.net).
3D-Chat Active Worlds
Active Worlds ist ein 3D Chat in dem Sie die Möglickeit haben, sich virtuell als Avatar in verschiedenen Welten zu bewegen. Hier können Sie Menschen aus der ganzen Welt kennenlernen, mit Ihnen virtuelle Häuser-, Grundstücke oder sogar Welten bebauen und dabei Spaß haben oder einfach ihrer Fantasie freien lauf lassen. Um in "Active World" teilzunehmen, laden Sie sich den aktuellen Browser herunter, wählen einen User - Namen aus und geben Ihre E-Mail Adresse ein und los gehts.
