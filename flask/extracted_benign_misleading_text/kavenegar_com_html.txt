وب سرویس پیامک
با ایجاد حساب، ۱۰,۰۰۰ ریال اعتبار هدیه دریافت کنید تا کیفیت وب سرویس پیامک و سرویس تماس اینترنتی را آزمایش کنید، وبسرویس پیام کوتاه کاوه نگار مورد اعتماد بیش از ۲۰۰۰۰ مشتری طی ۷ سال فعالیت بوده.
-
سرویس اعتبار سنجی
سرویس اعتبارسنجی امکان ارسال کد فعال سازی (کد ورود دو مرحله 2FA ) یا ارسال کد OTP و یا پیام های حساس سیستمی را از طریق وب سرویس ارسال پیامک و تماس صوتی به کلیه خطوط موبایل در ۱۴۸ کشور فراهم میکند.
-
وب سرویس تماس اینترنتی
این سرویس امکان برقراری تماس صوتی با پلتفرم های وب و اندروید و iOS را در اپلیکیشن شما فراهم می کند در واقع بدون نیاز به بستر مخابرات و فقط از طریق SDK کاوه نگار قابلیت تماس اینترنتی امن را به سرویس خود اضافه کنید.

چرا وب سرویس کاوه نگار
کاوه نگار سرویس ارسال و دریافت پیامک را از فراهم کننده های اصلی مخابرات تهیه نموده و آن را در غالب ساختار وبسرویس REST و SOAP با رعایت اصول طراحی API پیاده سازی کرده است.
حال این موضوع مطرح میشود که چرا وب سرویس کاوه نگار در مقابل سرویس های مشابه و حتی فراهم کننده اصلی؟معرفی وب سرویس پیامک

وب سرویس پایدار Availability
وبسرویس پیام کوتاه به لحاظ نوع عملکرد از حساسیت زیادی برخوردار است بنابراین پایداری و حفظ کیفیت مطلوب برای سرویس های API یک مزیت محسوب می شود، شما می توانید وضعیت سرویس های کاوه نگار را از Status Page بررسی نمائید.
-
جایگزینی خودکار و پوشش Redundancy
همکاری با کلیه سرشماره ها و قابلیت تعیین خط جایگزین در شرایط اختلال می تواند تضمین رسیدن پیامک های حساس شما با حداکثر سرعت به گیرندگان باشد ، وب سرویس OTP کاوه نگار رسیدن پیام به گیرنده را در کمتر ۱ دقیقه گارانتی میکند .
-
وب سرویسREST API
ساختار RESTful برای API پیامک امکان ارسال و دریافت پیامک را در پلتفرم های مختلف امکان پذیر می کند، کاوه نگار بصورت تخصصی بر روی API پیامک و تماس صوتی و ارسال کد تایید هویت Verification تمرکز دارد.
-
کیت توسعه سریعSDK
دیگر نیازی نیست بامستندات سر و کله بزنید، با وجود کیت توسعه سریع (SDK) برای زبان های C# , Node Js , PHP , Go , Ruby, Python, Java کافیست به صفحه نمونه کد ارسال بروید ، برای مشاهده نمونه های بیشتر به گیت هاب کاوه نگار سری بزنید.
-
خدمات تخصصیSpecialization
درگاه ارسال و دریافت پیامک گاوه نگار برای بستر سیستم های نرم افزاری طراحی شده است و با توجه به ساختار REST API کاوه نگار امکان ارسال و دریافت پیامک در پلتفرم های مختلف امکان پذیر است ، سرویس کاوه نگار به صورت تخصصی بروی ارائه وب سرویس پیامک فعالیت دارد .

درباره ما چه می گویند
CTO فیدیلیو
اغلب سرویس های ارسال پیامک در زمینه وب سرویس دچار ضعف های متعددی هستند، کاوه نگار با REST API قوی از رقبا جلوتر است. ما در فیدیلیو و دلیون نیاز خود را با وبسرویس های کاوه نگار کاملا برطرف کرده ایم.
CTO الو پیک
کاوه نگار با سرویس اعتبار سنجی و پشتیبانی سریع و همچنین سرویس API با اپ تایم بالا و قابلیت جایگزینی پیش شماره ها در شرایط اختلال باعث بهبود کیفیت سرویس الوپیک شده است.
مدیر تجربه مشتری تپ سی
ما در تپسی برای مدیریت ارسال پیام ها به مشتریان، نیاز به یک سامانه یکپارچه با امکان ایجاد کاربرهای متعدد با سطوح دسترسی قابلتنظیم داشتیم و از این نظر کاوهنگار با تیم فنی قوی و پشتیبانی پاسخگو بهترین گزینه موجود برای ما بوده است.
ویژگی های سرویس کاوه نگار
وب سرویس پیامک
اگر شما مایل باشید کیفیت و سرعت وب سرویس ارسال SMS را قبل از خرید تست کنید کافیست ثبت کنید و از اعتبار رایگان ۱۰،۰۰۰ ریال استفاده کنید، همچنین استفاده از نسخه استاندارد وب سرویس پیامک رایگان می باشد .
وب سرویس تماس اینترنتی
سرویس تماس (In-App Calling) به شما امکان میدهد تا تماس تلفنی را به محصول نرم افزاری خود به راحتی اضافه کنید ، برای شروع می توانید سرویس آزمایشی رایگان آن را با ثبت نام فعال کنید و کمتر از ۱ ساعت آن را به محصول خود اضافه کنید .
وب سرویس پیامک بین المللی
ارسال و دریافت پیامک به ۱۴۸ کشور دنیا، خیال شما را از بابت ارسال پیامک بین المللی به مشتریان در کشورهای مختلف راحت می کند همچنین امکان خرید خط اختصاصی از قاره اروپا و امریکا در سرویس کاوه نگار وجود دارد .
مدیریت مشتریان
امکان ساخت مشتری به عنوان زیر کاربر تحت اختیار شما و تخصیص حساب کاربری مجزا به آنها با اعتبار و تعرفه متفاوت، شما را از درگیری با این موضوع خلاص می کند ، برای اطلاعات بیشتر به صفحه مدیریت مشتریان مراجعه کنید .
ارسال به (Webhook) URL
ارائه Webhook برای دریافت (ارسال پیامک به Url) و وضعیت پیامک های ارسالی تعداد فراخوانی شما را بسیار کاهش می دهد و درواقع ساختار دریافت پیامک و دلیوری های شما را Real-Time می کند .
وب سرویس تماس صوتی
استفاده از وب سرویس تماس صوتی با قابلیت تبدیل متن به گفتار از طریق کنسول وب یا سرویس API راه ارتباطی دیگری برای تماس با مشتریان به شما می دهد.
سرویس اعتبارسنجی
سرویس ارسال OTP پیام مهم و حساس شما را از طریق API پیامک و تماس صوتی برای عملیات Verification در کمتر از ۱ دقیقه به دست گیرندگان میرساند.
افزونه ها Extensions
به راحتی افزونه ارسال پیامک ورد پرس جوملا مجنتو اپن کارت و... را به سیستم های خود اضافه کنید تا بیشتر با کاربران در ارتباط موثر باشید .
کاربرد وب سرویس SMS و تماس صوتی در صنعت IT
افزایش امنیت
سرویس های Lookup برای احزار هویت مشتریان با استفاده از کد های ورود دو مرحله 2FA و یا ارسال کد های OTP که باعث افزایش امنیت حساب های کاربر میشود، در این روش بعد از ورود ، کدی برای کاربر پیامک میشود تا در مرحله دوم کد را وارد نماید.
ارتباط با نرم افزارهای جانبی
وب سرویس ارسال و دریافت پیامک برای ارتباطات با دستگاه های مختلف در نرم افزار های مانیتورینگ (PRTG,Zabix) و مدیریت ارتباط با مشتری مانند (Dynamic CRM) همچنین امکان استفاده از سرویس مانیتورینگ نت نگار برای مانیتورینگ آپ تایم و ارسال پیامک یا تماس صوتی در شرایطی که شما به اینترنت دسترسی ندارید .
وب سرویس تماس صوتی
با استفاده از وب سرویس تماس صوتی شما می توانید برای بر قراری ارتباط با مشتریان که شماره ثابت دارند ویا درشرایطی که پیامک باز خورد مناسبی نداشته باشد، پیام خود را به آنها برسانید . اعلام میزان بدهکاری یا یادآوری یا انتقال کدهای OTP حساس و تایید شماره تلفن و شماره همراه برخی از کاربردهای تماس صوتی کاوه نگار هستند.
