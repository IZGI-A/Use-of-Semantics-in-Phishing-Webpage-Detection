Une nouvelle snapshot est sortie ce mercredi, la snapshot 19w41a. Cette snapshot inclue de nombreuses nouveautés: la majorité des nouveautés annoncées durant la Minecon pour la version 1.15. Découvrez le test détaillé de toutes les nouveautés apportées par cette snapshot dans cette news.
Une nouvelle snapshot est sortie ce mercredi, la version Minecraft Snapshot 19w40a. Cette version se concentre sur la correction de bugs, principalement des bugs introduit par le nouveau moteur Blaze 3D que nous avons découvert dans les précédentes snapshots.
