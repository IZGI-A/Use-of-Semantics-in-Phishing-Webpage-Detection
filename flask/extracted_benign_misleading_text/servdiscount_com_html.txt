Häufige Fragen zu unseren Dedicated Servern
-
Was ist ein dedicated Server?
Dedicated Server dienen exakt einem bestimmten Zweck. Der Server ist für einen Netzwerkdienst oder mehrere dauerhaft betriebene Dienste vorgesehen. Beim dedizierten Hosting wird die Hardware für nur einen Kunden bereitgestellt. Beim Shared Hosting hingegen werden viele einzelne Webauftritte auf einem physikalischen Server (Hardware) gebündelt. Ein Dedizierter Server empfiehlt sich dann, wenn ein Webprojekt die entsprechenden Ressourcen, welche vom jeweiligen Server bereitgestellt werden, allesamt benötigt. Jedoch sind die Anbindung an das Internet, die Prozessorleistung und die Größe des RAM-Speichers am bedeutendsten. Gerade weil diese Ressourcen nicht geteilt werden, stehen dem betroffenen Projekt genug Reserven zur Verfügung. Diese Reserven werden benötigt, wenn zeitgleich sehr viele Zugriffe stattfinden und der Server nicht nur Webseiten anzeigen muss, sondern gleichzeitig mehrere Datenbankabfragen und rechnerische Ausgaben bewältigen soll.
-
Wo ist der Unterschied zwischen einem Server mit Hardware-Raid und Software-Raid?
Zunächst einmal steht RAID für Redundant Array of Independent Discs, was übersetzt bedeutet Redundante Anordnung unabhängiger Festplatten. Damit man RAID verwenden kann, müssen zwei Festplatten Bestandteil eines dedicated Servers sein.
Durch RAID kann man folgende Vorteile erhalten:
- Erhöhung der Ausfallsicherheit durch Spiegelung der Festplatten
- Höhere Performance aufgrund des schnelleren Zugriffs auf die Festplatten
- Defekte Platten können während des Betriebs jederzeit ausgetauscht werden
- Einfache Verwaltung
Durch einen Hardware RAID-Controller kommen Sie in den Vorteil, dass Ihr System nicht belastet wird beim Spiegeln der Festplatten, denn dies übernimmt die Hardware für Ihren dedicated Server. Somit wird der Server mit RAID-Controller, seine CPU und vor allem Rechenleistung entlastet.
Beim Software RAID wird das spiegeln über das System bewältigt und keine zusätzliche Hardware benötigt. Zwar wird dadurch die Leistung des dedicated Server verringert, jedoch ist diese Variante kostengünstiger als ein dedicated Server mit Hardware RAID-Controller.
