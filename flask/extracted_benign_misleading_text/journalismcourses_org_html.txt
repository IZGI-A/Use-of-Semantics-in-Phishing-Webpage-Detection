Hands-on machine learning solutions for for journalists
Instructor: John Keefe
Dates: November 18 - December 15, 2019
Welcome to the Knight Center's new BOC, "Hands-on Machine Learning Solutions for Journalists"! During this four-week course, you'll learn how machine learning can enhance your reporting and the way you think and work as a journalist. We’ll hold your hand step-by-step through the concepts and code you’ll need to get a feel for using machine learning for journalism.
Data Journalism and Visualization
with Free Tools
Instructors: Alberto Cairo, Simon Rogers, Debra Anderson, Duncan Clark, Jan Diehm, Minhaz Kazi, Dale Markowitz,
Marco Túlio Pires, and Katherine Riley
Dates: October 14 - November 24, 2019
Welcome to the Knight Center's new MOOC, "Data Journalism and Visualization with Free Tools"! During this six-week course, which is powered by Google News Initiative, you'll learn about an array of free tools that can help improve your work in data journalism and visualization. By the end of the course, you will have learned how to find and get data, prepare data, find stories in data, visualize data and more.
