Cookies are small size dimension text strings, which the websites visited by the user send to their terminal (usually the browser), where they are stored in order to be re-transmitted to the same sites at the next visit by the same user.
Kaleyra uses technical cookies, which are those used only for carrying out the transmission of a communication over an electronic communication network, or which are strictly required by for a suppliers from an information society in which it is explicitly requested by the subscriber or by the user to provide the service.
Particularly Kaleyra uses analytics cookies which are needed to collect information on the number of users and how they visit the website.
In some cases, cookies that are related to information such as credentials entered for the access to restricted areas (email address and password), can be used to profile the user (personal habits, websites visited, downloaded content, types interactions carried out, etc.).
