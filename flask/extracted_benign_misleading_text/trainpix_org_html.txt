24.12.2018: Ночью на 24 декабря 2018 года произошла поломка базы данных, вследствие чего потеряны данные пользователей начиная с 12.02.2017. В данный момент восстановлены учетные записи для новых пользователей с 12.02.2017. Для пользователей, зарегистрированных раньше, и которые меняли любые данные в период между 12.02.2017 и 24.12.2018, необходимо внести эти изменения повторно. Для пользователей с регистрацией после 12.02.2017, необходимо восстановить пароль с помощью ссылки: https://trainpix.org/forgot.php По любым вопросам пишите на почту admin@trainpix.org, либо телеграм @kaliha
