Nous avons choisi d'adhérer au programme de sécurisation des paiements sur Internet (Verified by Visa ou SecureCode Mastercard).
Si votre banque adhère aussi à ce programme de sécurisation, vous verrez un nouvel écran s'afficher lors de vos paiements.
Vous trouverez toutes les instructions nécessaires sur cet écran : lisez-les attentivement !
Ce nouvel écran au logo de votre banque permet de saisir un code d'authentification personnel pour valider vos paiements.Ce code d'authentification varie selon les banques : date de naissance, code spécifique d'achat sur internet, code envoyé par SMS, etc.Ce code d'authentification vous a été communiqué par votre banque, soit sur votre relevé de compte, soit par SMS la plupart du temps.
Si vous ne connaissez pas ce code, un simple appel téléphonique à votre banque devrait résoudre le problème.
Attention
: Ce code d'authentification 3D Secure ne doit être confondu ni avec le code secret de votre carte bancaire, ni avec le cryptogramme visuel (3 derniers chiffres au dos de la carte).
